# Splash Screen Logo Setup

This app is configured to use a custom splash screen logo via Expo.

Expected file path:

`assets/dojo-logo.png`

Recommended:
- Format: **PNG**
- Size: **1024Ã—1024** (square) or higher
- Background: transparent or solid dojo brand color
- Content: centered dojo logo (no extra text if possible)

After placing the file, run Expo normally:

```bash
cd karate-dojo-mobile
npx expo start
```

Expo will use this image for the native splash screen as configured in `app.json`.
